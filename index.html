Website Builder Guide

1. Getting Started (Setup)

Open Landing Page:
Open index.html in Chrome, Edge, or Firefox.
Start Building:
Click "Start Building" → Opens biuld.html
Required Files:
  • index.html
  • biuld.html
  • style.css
  • server.js
  • images / videos (optional)

2. Builder Interface

Left Panel

Components Library

Center Panel

Drag & Drop Canvas

Right Panel

Properties Editor

3. How To Build

Step 1: Drag Components to Canvas
Step 2: Click Element to Edit Properties
Available Options:
  • Text Color
  • Background Color
  • Padding
  • Margin
  • Border Radius
  • Delete Element (❌ / Backspace)

4. Advanced Features

Canvas Background Image URL
Logo Upload Support
Navigation Editing
Live Property Updates

5. Export Website

Click Export → Copy generated index.html and style.css
Save files and open index.html in browser.
Your website is ready!
